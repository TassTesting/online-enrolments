**getStudentUD**
----
	Returns a structure of standard UD data for a student, or extended UD data for the UD area specified in JSON format.

* **Version History:**

    TASS v48.0 - Method Added

* **Version:**

	1

* **Method:**

	`GET | POST`
  
* **Params:**

   **Required:**
 
	`studcode [string]` - Student Code                    

   **Optional:**

	`areacode [integer]` - UD Area Code

   **Conditional:**

	none
